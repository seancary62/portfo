Overview

Since the early 20th century, cinema has been one of the most fundamental ways that we share stories with each other. From raunchy comedies and super-heroic epics to historical fiction, there is a story in film for everyone. For me, historical dramas have always been of interest. Watching a movie about a long-gone event can feel like a glimpse into the past. However, these films are often highly dramatized, exaggerated, or inaccurate in the way they tell the story. Sometimes watching a movie about a particular event can feel like jumping to the 14th chapter of a book. It was for this reason that I decided to create a timeline of historical movies. The current version has over 624 movies spanning from the prehistoric to the classical era. Movies from around the globe are included, as well as remakes and alternative versions. Users can see the geographical location of each movie and data such as the credits can be found by hovering over the object in the timeline. By looking at movies from different countries and release years, we can see the ways in which stories can be told from different perspectives, as well as the values of the storytellers.

Jupyter Notebook, TMDB API, GeoPy library, Tableau

Methods

With a collection of movie lists, an API was used to pull movie data from an open source database with Python. The data was cleaned and displayed in a timeline format using Tableau. a geolocation library was also used to plot the locations of the movies to provide additional insights.
